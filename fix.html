Saltychat Fix

Da Saltychat sich je nach Internet Provider nicht mit dem Websocket verbinden kann,
kann man die Google DNS Server nutzen damit alles korrekt funktioniert.

Folgende DNS Server müssen gesetzt werden:
IPv4
8.8.8.8   8.8.4.4
IPv6
2001:4860:4860::8888   2001:4860:4860::8844
